🚀 웹사이트 배포 방법 (Cloudflare Pages 추천)

  1. GAS 웹앱 배포 — Apps Script → WebApp_addon.gs 코드 붙여넣기 → 저장 → 배포 → 새 배포 → 웹 앱 (실행: 나, 액세스: 모든 사용자) → URL 복사
  2. index.html 수정 — 파일 상단 GAS_WEBAPP_URL에 복사한 URL 붙여넣기
  3. Cloudflare Pages 배포 — pages.cloudflare.com → 로그인 → Create application → Pages → Direct Upload → index.html 업로드 → 완료
  4. 카카오 Adfit — 각 페이지 ad-slot div 안에 Adfit 코드 삽입 후 재배포

⚠️ 로컬 파일(file://)로 열면 GAS API 연결이 안 됩니다 — 반드시 Cloudflare Pages 같은 웹 서버를 통해 서빙해야 합니다.
🏗️ 아키텍처: GAS 웹앱 + CacheService (완전 무료)

방문자 → GAS 웹앱 → 캐시 HIT → JSON 즉시 반환
                    → 캐시 MISS → Sheets 읽기 → 캐시 저장(6시간) → 반환

✅ 동시 수천 명 처리  |  ✅ 완전 무료

헤더(섹터명)는 시트에서 동적으로 읽어오므로 시트에 섹터를 추가/삭제해도 웹사이트가 자동 반영됩니다.
🔄 캐시 관리

데이터는 6시간 자동 캐싱됩니다. 헤더 우상단에 "시트 읽은 시각"이 표시됩니다.
장 마감 후 즉시 반영 → Sheets 메뉴 → ⚙️ 통합 설정 → 🌐 웹앱 API 캐시 초기화
